# Outdoor

## 프로젝트 소개

생기 넘치는 쨍한 주황 컬러를 포인트 컬러로 활용하여 활동적인 분위기를 연출했습니다. 또한 전체적인 레이아웃에 큼직한 이미지를 활용하여 상쾌하고 시원한 느낌을 전달하고자 했습니다.

# 데모 사이트

## 주요 기능

 : 미디어 쿼리를 사용하여 반응형 홈페이지를 제작하였습니다. 특정구간으로 줄어들었을 때 3단 메뉴 아이콘이 생성되고, 아이콘 클릭 시 gnb 메뉴가 나타납니다
